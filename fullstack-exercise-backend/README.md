## Installation

```bash
$ npm install
```

## Running the app

```bash
# docker (recommended - includes debug configuration for vs code)
$ docker compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
